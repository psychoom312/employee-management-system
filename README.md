# 💼 Employee Management System (Java Mini Project)

A complete **GUI-based Employee Management System** developed using **Java Swing**, demonstrating core Java concepts like OOP, Collections, File Handling, and Multithreading.

---

## 🚀 Features

✔ Add Employee  
✔ Update Employee (using Threading)  
✔ Delete Employee  
✔ Search Employee  
✔ Display Records in Table (JTable)  
✔ Sort Employees by Salary  
✔ Apply Bonus to All Employees  
✔ Data Persistence using Text File  
✔ Dark-Themed Modern GUI  

---

## 🛠️ Technology Stack

- **Language:** Java  
- **GUI:** Java Swing  
- **Collections:** HashMap, HashSet  
- **File Handling:** BufferedReader, BufferedWriter  
- **Concepts Used:**  
  - Inheritance & Polymorphism  
  - Inner Class  
  - Exception Handling  
  - Threading  
  - Collection Framework
